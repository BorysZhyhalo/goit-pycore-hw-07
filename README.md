# Assistant Bot (CLI адресна книга)

Консольний чат-бот для керування контактами в адресній книзі:

- зберігає **кілька телефонів** для одного контакту
- зберігає **необов’язковий** день народження
- валідовує телефон і дату народження
- показує **найближчі дні народження на наступні 7 днів** (якщо дата випадає на вихідні — переносить привітання на понеділок)


## Вимоги
- Python **3.10+** (рекомендовано)
- Працює на Windows / macOS / Linux


## Структура проєкту

```text
goit-pycore-hw-07/
├─ bot/
│  ├─ __init__.py
│  ├─ addressbook.py
│  ├─ commands.py
│  ├─ cli.py
│  └─ errors.py
└─ README.md
```

- Запускайте з кореня проєкту (там, де є папка bot/):

```bash
python -m bot.cli
```

- Після запуску ви побачите:

**Welcome to the assistant bot!**
**Enter a command:**

- Команди
hello

- Виводить привітання.

Приклад:

hello
add [ім'я] [телефон]

- Додає новий контакт або додає телефон до існуючого контакту.

телефон має складатися рівно з 10 цифр

- Приклади:

```bash
add John 1234567890
add John 0987654321
change [ім'я] [старий телефон] [новий телефон]

```
- Змінює телефонний номер для вказаного контакту.
новий телефон має складатися рівно з 10 цифр
Приклад:
```bash
change John 1234567890 1111111111
phone [ім'я]
```

- Показує всі телефонні номери контакту.
Приклад:
- phone John
Формат виводу:
**1234567890; 0987654321**

- all Показує всі контакти в адресній книзі.
Приклад:
all
Формат виводу:
```bash
John: 1111111111; 0987654321
Alice: 2222222222
```

- add-birthday [ім'я] [DD.MM.YYYY]

Додає день народження до існуючого контакту.
дата має бути у форматі DD.MM.YYYY

Приклад:
```bash
add-birthday John 05.03.1990
show-birthday [ім'я]
```

- Показує день народження контакту.
Приклад:
show-birthday John

Формат виводу:
```bash
05.03.1990
birthdays
```
Показує список контактів, яких потрібно привітати протягом наступних 7 днів.

### Правила:

якщо день народження випадає на суботу/неділю, дата привітання переноситься на понеділок

виводиться ім'я і congratulation_date

Приклад:

- birthdays

Приклад результату:
```bash
Alice: 09.03.2026
Bob: 09.03.2026
```

**close / exit**

Закриває програму.

Приклади:

close
exit
Валідація та повідомлення про помилки

### Бот валідовує введені дані й повертає зрозумілі повідомлення:

Телефон має містити 10 цифр

Дата народження має бути DD.MM.YYYY

### Якщо не вистачає аргументів:

Not enough arguments.

### Якщо значення некоректні — показується точна причина:

Phone must be 10 digits

Invalid date format. Use DD.MM.YYYY

### Якщо контакту немає:

Contact not found.

### Якщо команда невідома:

Invalid command.

### Швидкий тест (сценарій)

Запустіть бот:

python -m bot.cli

Потім введіть команди по черзі:

```bash
hello
all
add John 1234567890
add John 0987654321
phone John
change John 1234567890 1111111111
add-birthday John 05.03.1990
show-birthday John
birthdays
exit
```

### Нотатки

День народження зберігається як тип date для коректних обчислень.

Контакти зберігаються в AddressBook (а не у звичайному словнику).
